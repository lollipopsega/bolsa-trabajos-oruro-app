# Bolsa de Trabajos Oruro

Repositorio publico para descargar la aplicacion Android de Bolsa de Trabajos Oruro.

## Descargar la aplicacion

Archivo APK:

`apk/BolsaTrabajosOruro-contacto-visible-v16.apk`

## Nota

Esta aplicacion esta en etapa de prueba y se ira actualizando mientras se mejora el proyecto.

Si Android muestra una advertencia al instalar, es normal porque esta APK aun no esta publicada en Play Store.
